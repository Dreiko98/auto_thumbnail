# Thumbnail Generator

Generador profesional de thumbnails de 1920x1080px para blogs y redes sociales. Gratis, rápido y sin registros.

## Características

- 🎨 Generación de thumbnails profesionales
- 📱 Interfaz responsive y moderna  
- 🖱️ Drag & drop para archivos
- ⚡ Procesamiento instantáneo en el navegador
- 📥 Descarga directa en PNG optimizado
- 🎯 Hasta 4 iconos personalizables
- ✨ Efectos de sombra profesionales

## Tecnologías

- HTML5 Canvas para procesamiento de imágenes
- JavaScript puro (sin frameworks)
- CSS3 con variables y grid layout
- Optimizado para SEO y velocidad
